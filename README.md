# 프로젝트 설명
간단하게 유저매니저 페이지에 신규버튼을 누르면 신규유저를 등록하는 모달이 등장한다. </br>
신규 유저를 등록하는 과정에서 사용된 라이브러리 react-hook-form을 정리 하고자 한다.
